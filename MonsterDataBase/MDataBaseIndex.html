■説明
現在のVersionでデータ的に実装されているモンスターの情報ページです。
基本的な見方は石あげとほぼ同じです。
Lv入力欄を変更することで自動的にLvに合わせたステータスが計算されます。
Lv入力対応の特定異常抵抗、元素属性抵抗に使用している計算式は以下のものです。

最終元素属性抵抗 = floor(MOB基礎元素抵抗 + (MOB知恵 / 20.0))
最終異常抵抗 = floor(MOB基礎異常抵抗 * (1 + ((MOB知恵 + MOB威厳) / 500.0)))
最終低下抵抗 = floor(MOB基礎低下抵抗 * (1 + ((MOB知恵 + MOB威厳) / 1000.0)))
最終呪い抵抗 = floor(MOB基礎呪い抵抗 * (1 + ((MOB知恵 + MOB威厳) / 1000.0)))
最終致命打抵抗 = floor(MOB基礎致命打抵抗 * (1 + ((MOB知恵 + MOB威厳) / 1000.0)))
最終決定打抵抗 = floor(MOB基礎決定打抵抗 * (1 + ((MOB知恵 + MOB威厳) / 1000.0)))
最終特定異常抵抗 = floor(MOB基礎特定異常抵抗 + (MOB異常抵抗 * ((MOB知恵 + MOB威厳) / 1000.0)))
基礎経験値 = floor(MOB基礎経験値 / 10) * (MOBLV + 4))

 特定異常とは(暗闇、毒、睡眠、フリーズ、コールド、スタン、石化、混乱、魅了)を指します。
また、MAPに配置されているMOBの最終属性抵抗値は、
「MAP別追加属性抵抗」が加えられた値が最終値となります。
異常抵抗の算出式は、現在使用している計算式から僅かに変更されている可能性があるとの情報があり、
鵜呑みにしないようお願いします。

特定異常抵抗の判明していなかった部分については、
「ヴェルダンディ様」、「愛読者A様」のご協力の元、検証結果を元にして配置させて頂きました。

基礎経験値に関する計算式は暫定的な値となります。
スフィアなし、PTなし、G補正なし、称号効果、アイテム、実、キット等すべての上昇補正を除いた基準値となります
