■用語説明
基本的な見方は石あげと同じです。
アイテム特性について … 各アイテムに付与されている特性フラグです。判明できている物のみ記載。

[NPC売却禁止] … NPCへの販売が不可能である事を示す
[使用不可] … アイテムを消費することができないことを示す
[銀行取引不可] … 銀行取引が不可能である事を示す
[破壊不可] … インベントリ内からアイテムを破壊することができないことを示す
[取引不可] … 他プレイヤーキャラクターとの取引が不可能である事を示す
[アイテムに対して] … インベントリ内でアイテムに対して使用することを示す
[ベルト着用可] … ベルトに装着できることを示す
[着用中使用可] … アイテムを装備した状態で特殊効果を使用することができることを示す
[錬成可能] … Nxユニークへの錬成、Nxユニークの再錬成が可能なことを示す
